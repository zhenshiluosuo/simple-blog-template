# simple-blog


## Project describe
```
简单的一个博客小项目，数据利用的是http://jsonplaceholder.typicode.com/提供的样板
主体分为博文预览和写博文功能
包含基本的分页，搜索，跳转对应博文单独页面，简单封装axios等功能
```
